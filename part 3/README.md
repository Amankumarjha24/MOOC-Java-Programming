## 📖 Overview
This repository contains my solutions and progress for **Part 3** of the University of Helsinki's Java Programming MOOC. This section marks a transition from basic syntax to managing complex data structures and understanding memory allocation in Java.



---

## 🎯 Key Learning Objectives
In Part 3, the focus shifts toward efficient data handling and the internal mechanics of the JVM:

* **Dynamic Data Structures:** Mastering `ArrayList` for flexible data storage.
* **Memory Management:** Understanding **Reference Types** and how objects are stored in the Heap vs. Stack.
* **Arrays:** Implementing fixed-size structures for performance-critical logic.
* **String Manipulation:** Advanced text processing and the immutability of Strings.
